# 🎯 Быстрый старт - Локальная разработка

## ⚡ За 30 секунд

```bash
# 1. Установить зависимости
npm install

# 2. Запустить серверы
npm run local:universal

# 3. Открыть в браузере
# Компьютер: http://localhost:3000
# Мобильный: http://[ваш-ip]:3000
```

## 🚀 Основные команды

| Команда | Описание |
|---------|----------|
| `npm run local:universal` | Запустить все серверы |
| `npm run local:status` | Проверить статус |
| `npm run test:devices start` | Тестирование устройств |
| `npm run test:devices qr` | QR код для мобильного |

## 📱 Мобильное тестирование

### Автоматический QR код
```bash
npm run test:devices qr
```

### Ручной ввод
1. Убедитесь, что телефон в той же Wi-Fi сети
2. Откройте браузер на телефоне
3. Введите адрес: `http://[ip-компьютера]:3000`

## 🔧 Режимы работы

### Универсальный (рекомендуется)
- ✅ Работает на компьютере и мобильном
- ✅ Автоматически запускает API
- ✅ Hot reload

### Только локальный
```bash
npm run localhost
```

### Только мобильный
```bash
npm run mobile:192
# или
npm run mobile:196
```

## 🌐 Адреса

- **Компьютер**: `http://localhost:3000`
- **Мобильный**: `http://[ваш-ip]:3000`
- **API**: `http://localhost:3001`

## 🛠️ Проверка статуса

```bash
# Быстрая проверка
npm run local:status

# Детальная проверка
npm run test:devices status
```

## 🐛 Быстрое решение проблем

### Порт занят
```bash
lsof -ti:3000 | xargs kill -9
lsof -ti:3001 | xargs kill -9
```

### Перезапуск
```bash
npm run test:devices stop
npm run test:devices start
```

### Очистка
```bash
npm run clean
npm install
```

## 📋 Чек-лист

- [ ] Node.js >= 18.0.0 установлен
- [ ] Зависимости установлены (`npm install`)
- [ ] Порт 3000 свободен
- [ ] Мобильное устройство в той же сети
- [ ] Брандмауэр не блокирует порт 3000

## 💡 Советы

1. **Используйте универсальный режим** для разработки
2. **QR код** упрощает доступ с мобильного
3. **Проверяйте статус** перед началом работы
4. **Останавливайте серверы** после работы
