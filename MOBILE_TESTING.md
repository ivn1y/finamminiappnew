# 📱 Мобильное тестирование

Этот проект настроен для тестирования на мобильных устройствах через локальную сеть.

## 🚀 Быстрый запуск

### Способ 1: Автоматический скрипт (рекомендуется)
```bash
npm run mobile
```

Этот скрипт автоматически:
- Определит ваш IP-адрес в локальной сети
- Запустит сервер с доступом извне
- Покажет URL для подключения с мобильного устройства

### Способ 2: Ручной запуск
```bash
npm run dev:mobile
```

Затем откройте в браузере мобильного устройства:
```
http://[ВАШ_IP]:3000
```

## 📋 Требования

1. **Wi-Fi подключение**: Компьютер и мобильное устройство должны быть подключены к одной Wi-Fi сети
2. **Node.js**: Версия 18.0.0 или выше
3. **Порты**: Убедитесь, что порт 3000 свободен

## 🔧 Определение IP-адреса

### macOS/Linux:
```bash
ifconfig | grep "inet " | grep -v 127.0.0.1
```

### Windows:
```bash
ipconfig | findstr "IPv4"
```

## 📱 Подключение с мобильного устройства

1. Запустите сервер командой `npm run mobile`
2. Скопируйте показанный IP-адрес
3. Откройте браузер на мобильном устройстве
4. Введите адрес в формате: `http://[IP]:3000`

## ⚠️ Важные замечания

- **Безопасность**: Этот режим предназначен только для локальной разработки
- **Firewall**: Убедитесь, что брандмауэр не блокирует порт 3000
- **Сеть**: Устройства должны быть в одной локальной сети

## 🐛 Решение проблем

### Сервер не доступен с мобильного устройства:
1. Проверьте, что устройства в одной Wi-Fi сети
2. Убедитесь, что брандмауэр не блокирует соединения
3. Попробуйте перезапустить роутер

### Ошибка "Connection refused":
1. Проверьте, что сервер запущен
2. Убедитесь, что используется правильный IP-адрес
3. Проверьте, что порт 3000 свободен

## 📞 Поддержка

Если у вас возникли проблемы, проверьте:
- Версию Node.js: `node --version`
- Статус порта: `lsof -i :3000` (macOS/Linux)
- Сетевое подключение: `ping [IP_АДРЕС]`
