# 🚀 Finam Collab Mini App - Локальная версия

## ⚡ Быстрый запуск

```bash
# 1. Установить зависимости (если еще не установлены)
npm install

# 2. Запустить приложение
npm run start-universal

# 3. Открыть в браузере
# 💻 Компьютер: http://localhost:3000
# 📱 Мобильный: http://192.168.1.3:3000 (IP может отличаться)
```

## 🧪 Тестирование

```bash
# Проверить работу приложения
npm run test-universal
```

## 📱 Доступные адреса

- **💻 Компьютер**: `http://localhost:3000`
- **📱 Мобильный**: `http://192.168.1.3:3000`
- **🔧 Mock API**: `http://localhost:3001`
