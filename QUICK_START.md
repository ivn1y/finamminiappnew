# 🚀 Быстрый запуск локальных версий

## 📱 Мобильная версия (192.168.1.196/192)

```bash
# Запуск мобильной версии
npm run mobile:196
```

**Доступные адреса:**
- `http://192.168.1.196:3000`
- `http://192.168.1.192:3000`

## 💻 Локальная версия (localhost:3000)

```bash
# Запуск локальной версии
npm run localhost
```

**Доступный адрес:**
- `http://localhost:3000`

## 🔧 Альтернативные команды

```bash
# Через скрипты
./start-mobile-196.sh    # Мобильная версия
./start-localhost.sh      # Локальная версия

# Прямой запуск Next.js
npm run dev:mobile        # Мобильная версия
npm run dev:local         # Локальная версия
```

## ⚠️ Важно

- Убедитесь, что порт 3000 свободен
- Для мобильного тестирования устройства должны быть в одной Wi-Fi сети
- Для остановки нажмите `Ctrl+C`
