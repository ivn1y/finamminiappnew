# 🚀 Finam Collab Mini App - Быстрый старт

## ⚡ Запуск за 30 секунд

```bash
# 1. Запустить приложение
./start-local.sh

# 2. Открыть в браузере
# 💻 Компьютер: http://localhost:3000
# 📱 Мобильный: http://192.168.1.3:3000
```

## 🧪 Тестирование

```bash
# Проверить работу
./test-local.sh
```

## 📱 Доступные адреса

- **💻 Компьютер**: `http://localhost:3000`
- **📱 Мобильный**: `http://192.168.1.3:3000`
- **🔧 Mock API**: `http://localhost:3001`

## 🎯 Тестовые данные

### QR коды:
- `TRADER_001` - получение бейджа (+100 XP)
- `CONTENT_001` - разблокировка контента (+25 XP)

### Тестовый пользователь:
- ID: `user_123`
- Роль: `trader`
- XP: 150

## 🔧 Команды

```bash
./start-local.sh        # Запуск
./test-local.sh         # Тестирование
npm run mock-api         # Только API
npm run dev:local        # Только ПК
npm run dev:mobile       # Только мобильный
```

## ⚠️ Требования

- Node.js 18.0.0+
- Wi-Fi сеть (для мобильного)
- Современный браузер

## 🆘 Если что-то не работает

```bash
# Остановить все
killall node

# Очистить кэш
npm run clean

# Перезапустить
./start-local.sh
```
