# 🎯 Быстрый старт - Локальная разработка

## 🚀 Команды для запуска

### 💻 Локальная разработка (localhost:3000)
```bash
npm run localhost          # Через Node.js скрипт
npm run start-localhost    # Через shell скрипт
./start-localhost.sh       # Прямой запуск shell скрипта
npm run local              # Прямой Next.js
```

### 📱 Мобильное тестирование

#### Для IP 192.168.1.192
```bash
npm run mobile:192         # Через Node.js скрипт
npm run start-mobile-192   # Через shell скрипт
./start-mobile-192.sh      # Прямой запуск shell скрипта
```

#### Для IP 192.168.1.196
```bash
npm run mobile:196         # Через Node.js скрипт
npm run start-mobile-196   # Через shell скрипт
./start-mobile-196.sh      # Прямой запуск shell скрипта
```

## 🌐 Доступные адреса

| Режим | Адрес | Описание |
|-------|-------|----------|
| **Локальный** | `http://localhost:3000` | Только на компьютере |
| **Мобильный 192** | `http://192.168.1.192:3000` | Для мобильных устройств |
| **Мобильный 196** | `http://192.168.1.196:3000` | Для мобильных устройств |
| **Авто IP** | `http://[ваш-ip]:3000` | Автоматически определяемый |

## ⚡ Рекомендации

- **Для разработки на компьютере**: `npm run localhost`
- **Для тестирования на мобилке**: `npm run mobile:192` или `npm run mobile:196`
- **Убедитесь**, что мобильное устройство в той же Wi-Fi сети!
