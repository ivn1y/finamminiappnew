# Референсы уровня

Эта папка содержит референсные изображения для компонента уровня (`level-progress-circle`).

## Структура папки

Рекомендуемые названия файлов:
- `level-border-gradient.png` или `level-border-gradient.svg` - референс обводки уровня
- `level-progress-fill.png` или `level-progress-fill.svg` - референс заполнения опыта (прогресс-бара)
- `level-complete.png` или `level-complete.svg` - полный вид уровня с обводками и прогресс-баром

## Как использовать

1. Добавьте референсные изображения в эту папку
2. Я смогу прочитать эти файлы и использовать их для точной реализации дизайна
3. Форматы: PNG, SVG, JPG - любые изображения, которые показывают ожидаемый вид компонента
