# Telegram Bot для Mini App

Этот Python бот отправляет приветственное сообщение с кнопкой для запуска мини-приложения после команды `/start`.

## Настройка

1. **Установите Python зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Создайте бота в Telegram:**
   - Откройте [@BotFather](https://t.me/botfather) в Telegram
   - Отправьте команду `/newbot`
   - Следуйте инструкциям для создания бота
   - Сохраните полученный токен

3. **Настройте переменные окружения:**
   - Создайте файл `.env` в папке `telegram-bot/`
   - Добавьте следующие переменные:
   ```
   TELEGRAM_BOT_TOKEN=ваш_токен_от_BotFather
   MINI_APP_URL=https://ваш-мини-приложение-url.com
   ```

4. **Запустите бота:**
   ```bash
   python telegram-bot/bot.py
   ```

## Функциональность

- **Команда /start:** Отправляет приветственное сообщение с кнопкой "🚀 Поехали!"
- **Команда /help:** Показывает справку по использованию бота
- **Кнопка "Поехали!":** Запускает мини-приложение в Telegram

## Структура сообщения

```
Привет, [Имя]! 👋

Наше мини приложение для конференции TradeId поможет найти:
• Посмотреть карту и расписание
• Проходить квесты и получать бонусы

Нажмите кнопку ниже, чтобы начать! 🚀

[🚀 Поехали!] <- кнопка для запуска мини-приложения
```

## Интеграция с Mini App

Бот использует Telegram Web App API для запуска мини-приложения. Убедитесь, что:

1. Ваше мини-приложение правильно настроено в BotFather
2. URL мини-приложения доступен и работает
3. Мини-приложение поддерживает Telegram Web App API
