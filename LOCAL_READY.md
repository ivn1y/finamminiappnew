Ваше приложение настроено для локального тестирования и готово к использованию на компьютере и мобильных устройствах!

## 🛠️ Как запустить

### Простой способ (рекомендуется)
```bash
npm start
```

### Альтернативные способы
```bash
# Через скрипт
./start-universal.sh

# Универсальный режим
npm run dev:universal
```

## 📱 Как подключиться

### На компьютере
Откройте браузер и перейдите по адресу:
```
http://localhost:3000
```

### На мобильном устройстве
1. Убедитесь, что телефон и компьютер в одной Wi-Fi сети
2. Откройте браузер на телефоне и перейдите по адресу:
```
http://192.168.135.214:3000
```

### QR код для быстрого доступа
```bash
npm run qr
```

## 🔍 Проверка статуса

## 🐛 Решение проблем

### Если не можете подключиться с мобильного:
1. **Проверьте Wi-Fi**: Убедитесь, что устройства в одной сети
2. **Проверьте IP**: Запустите `npm run status` для получения актуального IP
3. **Проверьте брандмауэр**: Убедитесь, что порт 3000 не заблокирован
4. **Перезапустите сервер**: `Ctrl+C` и снова `npm start`

### Если страница не загружается:
1. Проверьте статус: `npm run status`
2. Попробуйте открыть `http://localhost:3000` на компьютере
3. Проверьте консоль браузера на ошибки
