# 🚀 Быстрый старт для тестирования

## ⚡ Одна команда для всего

```bash
npm run test-all
```

Эта команда запустит:
- ✅ Mock API сервер (порт 3001)
- ✅ Основное приложение (порт 3000)
- ✅ Поддержку мобильного тестирования

## 📱 Доступные адреса

После запуска `npm run test-all`:

- **Компьютер**: `http://localhost:3000`
- **Мобильный**: `http://[ваш-ip]:3000`
- **Mock API**: `http://localhost:3001`

## 🎯 Тестовые данные

### QR коды для сканирования:
- `TRADER_001` - получение бейджа
- `CONTENT_001` - разблокировка контента

### Тестовый пользователь:
- ID: `user_123`
- Роль: `trader`
- XP: 150

## 🔧 Альтернативные команды

```bash
# Только для компьютера
npm run test-local

# Только для мобильного
npm run test-mobile
```

## ⚠️ Важно

1. **Мобильное тестирование**: Убедитесь, что телефон в той же Wi-Fi сети
2. **Остановка**: Нажмите `Ctrl+C` для корректной остановки
3. **Порты**: Если порты заняты, скрипт автоматически их освободит

## 🆘 Проблемы?

Если что-то не работает:
1. Проверьте, что порты 3000 и 3001 свободны
2. Убедитесь, что мобильное устройство в той же сети
3. Перезапустите скрипт
