```

Откройте браузер на телефоне и перейдите по адресу:
```
http://192.168.1.10:3000

## 🐛 Решение проблем

### Если не можете подключиться:
1. **Проверьте Wi-Fi**: Убедитесь, что компьютер и телефон в одной сети
2. **Проверьте IP**: Запустите `npm run mobile` для получения актуального IP
3. **Проверьте брандмауэр**: Убедитесь, что порт 3000 не заблокирован
4. **Перезапустите сервер**: `Ctrl+C` и снова `npm run mobile`

### Если страница не загружается:
1. Проверьте, что сервер запущен: `lsof -i :3000`
2. Попробуйте открыть `http://localhost:3000` на компьютере
3. Проверьте консоль браузера на ошибки

## 📞 Поддержка

Если возникли проблемы:
1. Проверьте логи сервера в терминале
2. Убедитесь, что все зависимости установлены: `npm install`
3. Проверьте версию Node.js: `node --version` (должна быть >= 18.0.0)
