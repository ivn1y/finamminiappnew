# 🚀 Локальная версия для мобилки и компа

## ⚡ Быстрый запуск (рекомендуется)

```bash
npm run quick-start
```

**Или напрямую:**
```bash
./quick-start.sh
```

## 🔄 Стабильная версия с перезапуском

```bash
npm run auto-restart
```

**Или напрямую:**
```bash
./auto-restart.sh
```

## 🛠️ Полная версия с диагностикой

```bash
npm run start-local
```

**Или напрямую:**
```bash
./start-local.sh
```

## 📱 Доступные адреса

После запуска любого скрипта:

- **💻 Компьютер**: `http://localhost:3000`
- **📱 Мобильный**: `http://192.168.1.12:3000`
- **🔧 Mock API**: `http://localhost:3001`

## 🎯 Тестовые данные

### QR коды:
- `TRADER_001` - получение бейджа
- `CONTENT_001` - разблокировка контента

### Тестовый пользователь:
- ID: `user_123`
- Роль: `trader`
- XP: 150

## 🔧 Различия между скриптами

| Скрипт | Описание | Когда использовать |
|--------|----------|-------------------|
| `quick-start` | Самый простой запуск | Быстрое тестирование |
| `auto-restart` | С автоматическим перезапуском | Длительная разработка |
| `start-local` | Полная версия с диагностикой | Отладка проблем |

## ⚠️ Важно

1. **Мобильное тестирование**: Убедитесь, что телефон в той же Wi-Fi сети
2. **Остановка**: Нажмите `Ctrl+C` для корректной остановки
3. **Порты**: Скрипты автоматически освобождают занятые порты

## 🆘 Если что-то не работает

```bash
# Диагностика
npm run diagnose-mobile

# Очистка всех процессов
killall node

# Перезапуск
npm run quick-start
```

## 📋 Все доступные команды

```bash
npm run quick-start     # Быстрый запуск
npm run auto-restart    # С перезапуском
npm run start-local     # Полная версия
npm run diagnose-mobile # Диагностика
npm run test-all        # Универсальный тест
npm run test-local      # Только компьютер
npm run test-mobile     # Только мобильный
```
