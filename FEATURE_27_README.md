# Feature 27: Role Carousel Implementation

## Описание
Реализована карусель ролей для замены статичных кнопок выбора роли в процессе онбординга. Карусель позволяет пользователям перелистывать роли с видимыми соседними элементами на заднем фоне.

## Реализованные компоненты

### 1. RoleCarousel (`src/features/role-carousel/`)
- **Файлы**: 
  - `index.ts` - экспорт компонента
  - `ui/role-carousel.tsx` - основной компонент карусели

### 2. Интеграция с Onboarding
- Обновлен `src/entities/user/ui/onboarding.tsx`
- Заменена функция `renderRoleSelection()` на использование `RoleCarousel`

## Функциональность

### Основные возможности:
- **Карусель с тремя видимыми элементами**: центральный (активный) + два соседних (полупрозрачные)
- **Навигация**: кнопки "Назад/Вперед" и точки-индикаторы
- **Свайпы на мобильных**: поддержка touch-жестов для переключения ролей
- **Плавные анимации**: переходы с масштабированием и размытием
- **Адаптивный дизайн**: оптимизирован для мобильных устройств

### Визуальные эффекты:
- Центральная карточка: полная непрозрачность, нормальный размер
- Соседние карточки: 40% прозрачности, уменьшенный размер (75%), легкое размытие
- Плавные переходы: 500ms с easing функцией
- Hover эффекты для интерактивности

### Мобильная оптимизация:
- Touch-события для свайпов (минимальное расстояние 50px)
- Подсказка для пользователей о возможности свайпа
- Адаптивные размеры иконок и текста
