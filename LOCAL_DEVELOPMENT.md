# 🚀 Локальная разработка - Инструкции

## 📱 Мобильное тестирование

### Для IP 192.168.1.192
```bash
# Через npm скрипт
npm run mobile:192

# Через shell скрипт
./start-mobile-192.sh

# Через npm скрипт shell
npm run start-mobile-192
```

### Для IP 192.168.1.196
```bash
# Через npm скрипт
npm run mobile:196

# Через shell скрипт
./start-mobile-196.sh

# Через npm скрипт shell
npm run start-mobile-196
```

## 💻 Локальная разработка (localhost:3000)

```bash
# Через npm скрипт
npm run localhost

# Через shell скрипт
./start-localhost.sh

# Через npm скрипт shell
npm run start-localhost

# Прямой запуск Next.js
npm run local
```

## 🌐 Доступные адреса

### Мобильные версии
- `http://192.168.1.192:3000` - для IP 192
- `http://192.168.1.196:3000` - для IP 196
- `http://[ваш-ip]:3000` - автоматически определяемый IP
- `http://localhost:3000` - локальный доступ

### Локальная версия
- `http://localhost:3000` - только локальный доступ

## ⚠️ Важные замечания

1. **Мобильное тестирование**: Убедитесь, что мобильное устройство подключено к той же Wi-Fi сети
2. **Порт 3000**: Если порт занят, скрипты автоматически остановят конфликтующие процессы
3. **Hot Reload**: Все версии поддерживают автоматическую перезагрузку при изменении кода

## 🔧 Дополнительные команды

```bash
# Остановить все процессы на порту 3000
lsof -ti:3000 | xargs kill -9

# Проверить, что слушает порт 3000
lsof -Pi :3000 -sTCP:LISTEN

# Запустить mock API сервер
npm run mock-api
```
