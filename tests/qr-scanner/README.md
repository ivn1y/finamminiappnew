# QR Scanner Tests

Этот каталог содержит тесты для функциональности QR-сканера.

## Структура тестов

- `qr-utils.test.ts` - Тесты для утилит валидации QR-кодов
- `qr-api.test.ts` - Тесты для API сервиса QR-сканирования

## Запуск тестов

```bash
# Запуск всех тестов
npm test

# Запуск тестов QR-сканера
npm test -- --testPathPattern=qr-scanner

# Запуск тестов с покрытием
npm test -- --coverage --testPathPattern=qr-scanner
```

## Тестовые данные

Тесты используют следующие тестовые QR-коды:

- `FINAM:TRADER_2024` - Код трейдера
- `FINAM:STARTUP_ZONE_A` - Код стартапа
- `FINAM:EXPERT_MENTOR_2024` - Код эксперта
- `FINAM:PARTNER_BUSINESS` - Код партнера
- `FINAM:EVENT_MAIN_STAGE` - Код мероприятия
- `FINAM:ZONE_NETWORKING` - Код зоны
- `FINAM:DEMO_PITCH_2024` - Демо-код

## Интерактивное тестирование

Для интерактивного тестирования QR-сканера используйте страницу:
`http://localhost:3000/qr-test`

Эта страница позволяет:
- Открыть QR-сканер с камерой
- Тестировать ручной ввод кодов
- Просматривать историю сканирования
- Очищать историю
- Тестировать демо-коды
