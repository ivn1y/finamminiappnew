# 🚀 Finam Collab Mini App - Локальная разработка

## ⚡ Быстрый запуск

```bash
# 1. Установить зависимости (если еще не установлены)
npm install

# 2. Запустить приложение
npm run start-local-dev

# 3. Открыть в браузере
# 💻 Компьютер: http://localhost:3000
# 📱 Мобильный: http://192.168.1.XXX:3000 (IP будет показан в консоли)
```

## 📱 Тестирование на мобильном

1. Запустите сервер: `npm run start-local-dev`
2. Убедитесь, что мобильное устройство в той же Wi-Fi сети
3. Откройте IP-адрес из консоли на мобильном устройстве
4. Или отсканируйте QR-код из консоли

## 🛠️ Команды

```bash
npm run start-local-dev      # Запуск для ПК и мобильных устройств
npm run dev:local-universal  # Альтернативный запуск
npm run dev:local            # Только для компьютера
npm run dev:mobile           # Только для мобильных устройств
npm run status               # Проверка статуса серверов
```

## 📊 Доступные адреса

- **💻 Компьютер**: `http://localhost:3000`
- **📱 Мобильный**: `http://192.168.1.XXX:3000`
- **🔌 API сервер**: `http://localhost:3001`
