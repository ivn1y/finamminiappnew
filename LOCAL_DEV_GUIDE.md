# 🚀 Finam Collab Mini App - Локальная разработка

## ⚡ Быстрый старт

### 1. Запуск приложения
```bash
# Простой запуск
npm run start-local-dev

# Или через скрипт
./start-local-dev.sh
```

### 2. Доступ к приложению
- **💻 Компьютер**: `http://localhost:3000`
- **📱 Мобильный**: `http://192.168.1.XXX:3000` (IP будет показан в консоли)
- **🔌 API сервер**: `http://localhost:3001`

## 📱 Тестирование на мобильных устройствах

### Требования
- Мобильное устройство должно быть подключено к той же Wi-Fi сети, что и компьютер
- Современный браузер (Chrome, Safari, Firefox)

### Инструкции
1. Запустите сервер командой `npm run start-local-dev`
2. Скопируйте IP-адрес из консоли (например, `http://192.168.1.100:3000`)
3. Откройте этот адрес в браузере мобильного устройства
4. Или отсканируйте QR-код, который появится в консоли

## 🐛 Решение проблем

### Проблема: Не могу подключиться с мобильного устройства
**Решение:**
1. Убедитесь, что устройства в одной Wi-Fi сети
2. Проверьте IP-адрес в консоли
3. Отключите VPN на мобильном устройстве
4. Проверьте настройки файрвола

### Проблема: Сервер не запускается
**Решение:**
1. Проверьте версию Node.js: `node --version`
2. Установите зависимости: `npm install`
3. Очистите кэш: `npm run clean`
4. Перезапустите терминал

### Проблема: Порт занят
**Решение:**
```bash
# Остановить процессы на портах
lsof -ti:3000 | xargs kill -9
lsof -ti:3001 | xargs kill -9
```

Скрипт автоматически:

## 📈 Мониторинг

### Файл статуса
Создается файл `local-dev-status.json` с информацией о:
- Статусе серверов
- Сетевых адресах
- Времени запуска
- Конфигурации

### Логи
Все логи выводятся в консоль с цветовой индикацией:
- 🟢 Зеленый - успешные операции
- 🟡 Желтый - предупреждения
- 🔴 Красный - ошибки
- 🔵 Синий - информация
